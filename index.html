Features

Todo para escribir menos, hacer más

Cinco tipos de atajos que eliminan el 90% de lo que repetís todos los días.

💬

Textos rápidos

Escribís /firma, /ok, /follow y se reemplaza por un texto completo. En cualquier app.

🔍

Search Shortcuts

Seleccioná una dirección → Maps. Un nombre → LinkedIn. Un producto → Amazon.

⌨️

Snippets con hotkeys

Ctrl+Shift+F → firma. Ctrl+Shift+R → respuesta rápida. Pegan texto al instante.

🚀

Lanzador de apps

Abrí WhatsApp, Calendar, Notion o cualquier URL con un atajo. Sin tocar el mouse.

📋

Templates incluidos

16 atajos preconfigurados + catálogo de templates: emails, ventas, soporte, redes.

📥

Import / Export

Compartí tus atajos con tu equipo en JSON. Importá kits pre-armados por rol.

Cómo funciona

3 pasos. 10 segundos.

01

Elegí el tipo

"Texto rápido" para triggers o "Atajo de teclado" para hotkeys. Sin código, todo visual.

02

Configurá

Ponele un trigger (/firma) o una combinación (Ctrl+Shift+F) y el texto o acción.

03

Usalo donde sea

Gmail, WhatsApp, Word, Slack — funciona en cualquier app de tu sistema.

FAQ

Preguntas frecuentes

¿Es realmente gratis? ▾
Sí, 100% gratis, sin límites, para siempre. El código fuente está en GitHub para que cualquiera pueda auditarlo.
¿Necesito saber programar? ▾
No. Todo es visual. Elegís tipo de atajo, completás datos, guardás. Si sabés llenar un formulario, sabés usar NewenKey.
¿Funciona en Mac y Windows? ▾
Sí. Windows 10/11 y macOS (Intel y Apple Silicon). Los textos rápidos con /trigger funcionan en Windows. En Mac funcionan hotkeys, snippets, búsquedas y lanzador.
¿Funciona en cualquier programa? ▾
Sí. Los atajos son a nivel de sistema operativo. Gmail, Word, WhatsApp Web, Slack, VS Code — donde estés, el atajo se activa.
¿Mis datos se suben a algún lado? ▾
No. Todo queda en tu máquina. NewenKey no se conecta a internet. Cero telemetría, cero tracking.
¿Qué significa "Newen"? ▾
"Newen" significa fuerza, energía y poder en mapudungun, la lengua del pueblo mapuche. Cada atajo que creás canaliza tu newen digital.
¿El antivirus lo detecta como virus? ▾
Algunos antivirus alertan sobre apps nuevas sin firma digital. Es un falso positivo. Podés verificar el código fuente en GitHub.
