# ⚡ NewenKey · by gbull

**Atajos de teclado y textos rápidos, sin código.**

Escribí `/firma` y se reemplaza solo. Seleccioná texto y buscalo en Maps al instante. Todo visual, todo rápido.

> *"Newen" significa fuerza, energía y poder en mapudungun, la lengua del pueblo mapuche.*

## ⚡ Features

- **Textos rápidos** — Escribís /firma, /ok, /follow y se reemplaza automáticamente
- **Hotkeys globales** — Ctrl+Shift+G → busca en Google, Ctrl+Shift+M → busca en Maps
- **Search Shortcuts** — Seleccioná texto y buscalo en Google, Maps, YouTube, Amazon, LinkedIn
- **Lanzador de apps** — Abrí WhatsApp, Calendar, Notion con un atajo
- **16 atajos preinstalados** — Firma, presentación, follow up, datos bancarios, redes
- **Templates por categoría** — Emails, ventas, soporte, dev, redes
- **Import/Export JSON** — Compartí packs de atajos con tu equipo
- **100% offline** — Sin cuenta, sin nube, sin telemetría

---

## 🔒 Seguridad

- **Zero network** — No hace ningún request a internet
- **Zero telemetría** — No se trackea nada
- **Local-only** — Tus atajos se guardan en tu máquina, nunca salen de tu PC
- **Código auditable** — Todo el código fuente está acá

## 🔧 Build desde el código fuente
```bash
git clone https://github.com/GonzaloBullian/newenkey.git
cd newenkey
npm install
npm run dev
```

Build Windows: `npm run package:win`
Build macOS: `npm run package:mac`
